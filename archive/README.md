# Archive

This directory contains archived files from the project that are no longer actively used but kept for historical reference.

## python-scripts/
One-off Python scripts used for data conversion and JSON fixing during early development:
- JSON quote fixing utilities
- Data set conversion scripts
- Backup and rebuild scripts

These scripts were used during the initial setup and data migration phases.

## old-docs/
Deprecated documentation and planning files from earlier iterations:
- Refactoring plans
- Feature specifications
- Navigation implementation guides
- Master specifications

These documents have been superseded by current documentation in the root directory (CLAUDE.md, CHANGELOG.md, README.md).
