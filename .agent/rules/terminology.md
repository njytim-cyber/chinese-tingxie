# Terminology

- **听写 (Tīngxiě)**: Spelling / Listening. Users listen to a word and write it.
- **默写 (Mòxiě)**: Dictation. Users listen to a longer passage/sentence and write it.

**Validation Rule**: Always check if the user is referring to the correct mode based on these Chinese terms.
